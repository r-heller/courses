# dag-explorer

Companion to **Course 3, Week 3, Session 3 — DAGs with dagitty.** Pick
a pre-built DAG (confounder, mediator, collider, M-bias) or paste your
own in `dagitty` syntax. The app computes minimal sufficient
adjustment sets via the back-door criterion and lists the implied
conditional independencies.

Packages: `shiny`, `dagitty`, `ggdag`, `ggplot2`.

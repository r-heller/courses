# diagnostic-test

Companion to **Course 1, Week 2, Session 3 — Diagnostic testing.**
Sliders for sensitivity, specificity, and disease prevalence; outputs
the per-1000 outcome table, PPV, NPV, LR+, LR-, and a PPV-vs-prevalence
curve. The app makes Bayes' theorem visible in the way that surprises
clinicians most: a high-sensitivity test on a low-prevalence population
still produces lots of false positives.

Packages: `shiny`, `ggplot2`.

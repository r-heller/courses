# anova-contrasts

Companion to **Course 2, Week 2, Session 1 — One-way ANOVA and
contrasts.** Pick the number of groups, sample size per group, the
spread of true means, and within-group noise. The app shows the
boxplot, the ANOVA table, and Tukey HSD pairwise comparisons with
adjusted p-values.

Packages: `shiny`, `ggplot2`, `broom`.

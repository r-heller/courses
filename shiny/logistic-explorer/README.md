# logistic-explorer

Companion to **Course 2, Week 3, Sessions 1 and 5 — Logistic regression
+ calibration / discrimination.** Pick intercept, slope, and X spread.
The app simulates the data, fits the logistic model, and shows the
fitted curve, ROC, and decile-binned calibration plot, with AUC and
Brier score.

Packages: `shiny`, `ggplot2`, `pROC`.

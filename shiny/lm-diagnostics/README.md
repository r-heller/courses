# lm-diagnostics

Companion to **Course 2, Week 1, Session 4 — Diagnostics: residuals,
leverage, Cook, VIF.** Generate data with chosen slope, noise level,
optional heteroscedasticity, and an optional high-leverage outlier;
inspect the four diagnostic plots side-by-side. The fastest way to
build intuition for what each plot is telling you.

Packages: `shiny`, `ggplot2`, `broom`.
